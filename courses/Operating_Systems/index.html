Operating Systems MC4109 3 Autumn 2024

About:

This course provides an introduction to fundamental concepts of operating systems and describes concepts of operating systems in terms of functions, structure, and implementation. It covers the principles and techniques in the design of operating systems. Topics included in this course are process management,scheduling policies for CPU, inteprocess communications, synchronization, deadlocks, memory management, file management, and input/output operations.

Contents:

This course will cover fundamental concepts of operating systems.

  • Overview
    • Introduction to Operating Systems
    • History of Operating Systems, Introduction to Unix/Linux, MS-DOS and Windows
    • Operating System Structures
  • Process Management
    • Processes and threads
    • Scheduling
    • Inter process communication
    • Synchronization and deadlocks
  • Memory Management
    • Memory allocation
    • Paging and segmentation
    • Virtual memory
  • File System Management
    • File system interface
    • File system implementation
  • IO Management
    • IO systems
    • Disk management
  • Security

Objective:

  • To understand the functions of operating systems;
  • To understand the concept of process , process management and process schduling;
  • To understand diffrence between user level programming and kernel level programming;
  • Understand and implement interprocess communication (IPC) and synchronization;
  • To understand different approaches to memory management and able to distinguish difference between main memory and virtual memory;
  • Students should be able to use system calls for managing processes, memory and the file system;
  • Explain process management, processor scheduling, concurrent programming, deadlocks and synchronization, memory management, file management and I/O systems, disk scheduling;

Assessment:

  • Class test(s):10%
  • Assignment(s): 20%
  • Mid-term written examination : 20%
  • End-term written examination: 50%

Prerequisites:

To undertake this course,You should also be familiar with the following:
  • C and C++ programming experience
  • Knowledge of Linux Operating Systems or similar OS (working in command line environment)
  • Knowledge of basic concepts in data structures and algorithms
